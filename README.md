# worm-game

지렁이 게임을 만들건데 일단 지렁이를 1cm로 잡고 지렁이가 들어갈 곳의 크기는 1mx1m로 해줘 글고 지렁이가 벽에 닿으면 죽게 만들고 경험치가 랜덤한곳에서 30초 마다 나오게 만들어줘 그리고 지렁이가 경험치를 먹으면 1cm씩 크게 만들어줘

## Tech stack

This project is built with React and Chakra UI.

- Vite
- React
- Chakra UI

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/worm-game.git
cd worm-game
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.

## Requirements

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
